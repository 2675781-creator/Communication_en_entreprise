# Communication en entreprise – Espace collaboratif GitHub

## Contexte du projet
Ce dépôt GitHub a été créé dans le cadre du cours Communication en entreprise. L’objectif est de mettre en place un espace collaboratif professionnel permettant de reproduire un flux de travail réel en entreprise, en utilisant GitHub comme outil central de communication, de collaboration et de gestion des versions.

Le projet s’appuie sur un travail technique déjà réalisé dans un autre cours, qui est ici structuré, documenté et amélioré afin de répondre à des standards professionnels.

## Objectifs du projet
- Mettre en place un espace collaboratif clair et structuré
- Travailler efficacement en équipe à l’aide de GitHub
- Appliquer de bonnes pratiques de gestion des versions
- Communiquer de manière professionnelle (documentation, commits)
- Valoriser le projet dans les portfolios numériques des membres de l’équipe

## Méthodologie et approche
L’équipe adopte une méthodologie inspirée des pratiques professionnelles :
- Utilisation d’un dépôt GitHub central
- Travail sur des branches individuelles par membre
- Utilisation de Pull Requests pour intégrer les modifications dans la branche principale (main)
- Messages de commits clairs, précis et cohérents

Cette approche permet d’assurer une bonne traçabilité du travail et une collaboration efficace.
